# Neon Pantry Panic

Arcade ingredient-guessing game built as a self-contained browser game.

Open `index.html` directly in a browser, or launch it from the root hub.

## Controls

- Click ingredient cards to choose pantry items.
- Number keys `1` through `9` choose visible ingredient cards.
- `M` toggles mute.
- `F` uses Freeze +10.
- `T` uses Toss Decoys.
- `S` uses Safety Plate.
- Chaos Mode adds louder wrong-answer effects.
- Solo Run starts a single-player service.
- 2 Teams alternates control; mistakes pass the turn.

## Content

- 6 categories.
- Full Service mode pulls eight random dishes from every station.
- 36 dishes.
- Each dish has required ingredients, decoys, difficulty, and custom wrong-answer lines.
- Service modifiers change the round rules.
- Limited pantry tools add mid-round choices.
- Coins are earned after runs and unlock additional stations.
- Station unlocks, coins, and high score persist in local storage.
- Uses procedural Web Audio and CSS/canvas effects only; no external frameworks or fetched assets.
